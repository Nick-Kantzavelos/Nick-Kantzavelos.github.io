# Windows Server Security Baseline

## Objective
Create a secure baseline for a Windows Server file server.

## Tools and Concepts
- Windows Server
- Access control
- Windows Defender Firewall
- Least privilege
- Validation procedures

## What I Did
I developed a baseline focused on securing a file server used to store and share organizational files. The baseline included role-based access control, firewall configuration, and validation steps to confirm the server was operating securely.

## Key Security Measures
- Restricted access to authorized users only
- Applied least privilege
- Enabled Windows Defender Firewall
- Defined validation steps for administrators

## Skills Demonstrated
- System hardening
- Security documentation
- Administrative baselining
- Technical communication
